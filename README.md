# Expense Tracker (Python)

A command-line personal expense tracker built with Python and SQLite. Add income and expenses, categorize transactions, generate monthly reports, and search past entries — all stored in a local database.

## Features
- Add income entries
- Add expense entries with categories
- View all transactions in a clean table format
- Generate monthly reports (total income, total expenses, net savings, category-wise breakdown)
- Search transactions by category, note, or date

## Tech Stack
- Python
- SQLite
